数据类型

数据类型介绍

  • 整型
    • byte占1字节，范围：-128 - 127
    • short占2字节，范围： -2^15 - 2^15-1
    • int占4字节，范围： -2^31 - 2^31 - 1
    • long占8字节，范围： -2^63 - 2^63-1
  • 浮点类型
    • float占4字节 -21024 ~ -2-1074 —— 0 —— 2-1074 ~ 21024
    • double占8字节
  • 字符类型：char占2字节
  • 布尔
    • boolean 只有两个值：true、false占1位。
  • 整数拓展-进制转换
    • 二进制前缀 0b 或 0B 例：0b1
    • 八进制前缀 0 例：01
    • 十六进制前缀 0x 或0X 例：0x1
  • 字符扩展-编码
    • 所有字符的本质还是数字 编码 Unicode 2字节 例： \u0061

类型转换

类型转换要注意：

  1. 不能对布尔类型转换
  2. 不能将对象类型转换为不相干的类型
  3. 在把高容量转换到低容量时，强制转换
  4. 转换的时候可能存在内存溢出，或者精度问题！
  • 低 ——–> 高
  • byte -> short -> char -> int -> long -> float -> double 
public class Demo3 {
    public static void main(String args[]) {
        int i=128 ;
        //byte b= byte(i);//强制类型转换 高-->低 超范围溢出会报错
        double b1 = i;
        // JDK7新特性，数字之间可以用下划线分割
        int money = 10_0000_0000;
        System.out.println();
    }
}

运算符

  • 数学函数与常量
    • Math.sqrt(x)：平方根
    • Math.pow(x, y)：幂运算，x 的 y 次幂
    • Math.floorMod(x, y)：求余
  • 自增与自减运算符
    • 建议不要在表达式中使用 ++，容易带来疑惑和 bug
    • int b = 1++; // b=2 && int b = ++1; // b=1
  • 关系和 boolean 运算符
    • ==
    • !=
    • <
    • >
    • <=
    • >=
    • &&（逻辑与：两个变量都为真，结果才为true）
    • ||（逻辑或：两个变量有一个为真，则结果才为true）
    • ?:
  • 位运算符
    简单理解，将值转为二进制进行同位取值比较
    • 整型
      • | (或)
      • & (且)
      • ~ (非)
      • ^ (抑或)
      • 布尔值
      • 可用 & 和 |，只是不短路
    • 位移
      • <<（左移）例：3<<1;3的二进制11 往左移动边为 110即6
      • >>（带符号右移）3>>1;3的二进制11 往右移动边为 1.1即1
      • “符号扩展”：若值为正，则在高位插入 0；若值为负，则在高位插入 1。
  • 三元运算: x ？ y : z如果x==true,则结果为y，否则结果为z

  • 设计方法的原则：方法的本意是功能块，就是实现某个功能的语句块的集合。我们设计方法的时候，最好保持方法的原子性，==就是一个方法只完成1个功能，这样利于我们后期的扩展==
  • 一个前提:方法在主类中定义,并且由主方法直接调用

修饰符

  • private：仅对本类可见
  • public：对所有类可见
  • 接口里方法默认 public
  • protected：对本包和所有子类可见
  • default：对本包可见
  • 一个方法可以访问所属类的所有对象的私有对象
  • final
    • final 域（常量）
    • final 域必须显式指定初始值，赋值后，不能被重新赋值。
    • 基本类型：final 使数值不变
    • 引用类型：final 使引用不变，但是被引用的对象本身是可以修改的
    • final 方法
    • final 方法可以被子类继承，但是不能被子类修改。
    • private 方法隐式地被指定为 final，如果在子类中定义的方法和基类中的一个 private 方法签名相同，此时子类的方法不是重写基类方法，而是在子类中定义了一个新的方法。
  • static关键字
    • 可以通过类名称直接调用。static属性虽然定义在类之中,但是其并不受到类实例化对象的控制(可以直接在类使用)。
    • static方法只允许调用static属性或static方法;
    • 非static方法允许调用static属性或static方法;

方法的重载

方法重载的定义:当方法名称相同,参数的类型或个数不同的时候就称为方法的重载，开发习惯方法重载返回值类型相同。 

public class Demo8 {
    public static int add(int a,int b){
        return a+b;
    }
    public static int add2(int a,int b,int c){
        return a+b+c;
    }
    public static void main(String[] args) {
        int sum = add(1, 2);
        System.out.println(sum);
        int sum2 = add2(1, 2, 3);
        System.out.println(sum2);
    }
}

递归

递归就是：A方法调用A方法！就是自己调用自己。

  • 递归结构包括两个部分：
    • 递归头：什么时候不调用自身方法。如果没有头，将陷入死循环；
    • 递归体：什么时候需要调用自身方法。 
public class Demo9 {
    // 5! 5*4*3*2*1
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.println("请输入一个数:");
        int number = scanner.nextInt();
        int test = test(number);
        System.out.println(number + "的阶乘:" + test);
    }
    public static int test(int n){
        if(n==1){
            return 1;
        }else{
            return n*test(n-1);
        }
    }
}
