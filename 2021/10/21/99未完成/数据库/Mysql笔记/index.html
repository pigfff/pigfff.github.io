MySQL基础

MySQL 是最流行的关系型数据库管理系统，在WEB应用方面 MySQL是最好的 RDBMS(Relational Database Management System：关系数据库管理系统)应用软件之一。

  • 数据以表格的形式出现
  • 每行为各种记录名称
  • 每列为记录名称所对应的数据域
  • 许多的行和列组成一张表单
  • 若干的表单组成database

  列名 列类型 列属性,
  ...);
//删

//查
select 【字段】 from 【表名】
  • UNION

UNION 操作符用于合并两个或多个 SELECT 语句的结果集。
请注意，UNION 内部的每个 SELECT 语句必须拥有相同数量的列。列也必须拥有相似的数据类型。同时，每个 SELECT 语句中的列的顺序必须相同。
SELECT column_name(s) FROM table1 UNION SELECT column_name(s) FROM table2;

操作符 描述 备注
= 等于
<> 不等于。 注释：在 SQL 的一些版本中，该操作符可被写成 !=
> 大于
< 小于
>= 大于等于
<= 小于等于
BETWEEN 在某个范围内 where id between 1 and 2
LIKE 搜索某种模式 ‘a%’: a开头如：a…….’%XYZ’: a结尾：…..a’%AN%’: 含有a。
IN 指定针对某个(具体)列的多个可能值 where id IN (1,3,…) 等于 id=1 or id =3 or id=…
  • ORDER BY 关键字用于对结果集按照一个列或者多个列进行排序

information_schema表

information_schema 用于存储数据库元数据(关于数据的数据)，例如数据库名、表名、列的数据类型、访问权限等。

select schema_name from information_schema.schemata;#等于show databases()
Select table_name from information_schema.tables where "库名字";#Select table_name from information_schema.tables where TABLE_schema='SECURITY'

Select column_name from information_schema.columns where table_schema='库名字' and table_name = '表名'#或者Select * from 库名.表名
