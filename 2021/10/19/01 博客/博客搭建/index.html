博客搭建

2021-10-19 22:16:12
Categories： Tags：

博客搭建—基于hexo和github

安装流程

本次安装是在mac环境下进行安装的
需要node.jd,官方网站可以直接进行下载
判断安装是否成功

node -v
npm -v

选择国内源下载cnpm 

npm install -g cnpm --registry=https://registry.npm.taobao.org

安装hexo 

cnpm install -g hexo-cli

创建博客文件夹 相关路径/bolg

mkdir bold

进行bolg文件夹子 初始化hexo init

sudo hexo init

开启博客

hexo s

清理缓存

hexo cl

生成

hexo g

此时本地部署完成，可以看到通过访问4000端口查看

部署到github上

下载git插件

cnpm install --save hexo-deployer-git

修改_config.yml文件，在当时创建的/blog文件夹里

#第16行修改你的博客
url: https://pigfff.github.io
#最后一行修改
deploy:
  type: 'git'
  repo: git@github.com:项目/pigfff.github.io.git
  branch: main

设置用户邮箱

git config --global user.name "用户名"
git config --global user.email "用户邮箱"

生成密钥

ssh-keygen -t rsa -C "用户邮箱"
#直接回车4次

#生成秘钥位置
/var/root/.ssh/id_rsa.pub

#并将密钥id_rsa.pub设置到github上
#路径：用户->setting->SSH and GPFkeys->New SSH key

此时项目配置完成进行部署到github上

hexo cl && hexo g && hexo d

deploy：repo 选择了ssh连接方式
brach main分支
说明：选择 ssh 配置公私密钥不用输入密码比较方便
要记得将项目main分支设置为默认

HEXO皮肤

HEXO皮肤可以去https://hexo.io/themes/进行挑选
此博客皮肤来源https://github.com/wujun234/hexo-theme-tree
安装过程注意导航栏和图标内themes/tree/layout/_partial要修改为自己博客路径

安装插件Markdown-it 

#在bolg文件夹中
npm install markdown-it --save

#修改blog文件夹_config.yml 
highlight:
  enable: false
  line_number: false
  auto_detect: false
  tab_replace: ''

一些补充

socks代理

git config --global http.proxy 'socks5://127.0.0.1:1080'
git config --global https.proxy 'socks5://127.0.0.1:1080'

也可以直接修改~/.gitconfig文件。

vi ~/.gitconfig

新建或修改这两项配置

[http]
proxy = socks5://127.0.0.1:1080
[https]
proxy = socks5://127.0.0.1:1080
